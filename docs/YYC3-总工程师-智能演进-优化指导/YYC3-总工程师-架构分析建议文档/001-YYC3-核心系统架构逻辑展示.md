# YYC³ Cloud Intelli-Matrix 核心系统架构逻辑展示

## 一、架构总览

### 1.1 系统定位

**YYC³ Cloud Intelli-Matrix** 是一款面向 AI 智能应用的全栈监控与运维平台，采用 **React 19 + TypeScript + Electron** 技术栈，融合 PWA 离线能力、实时数据推送、AI 智能助手等特性，构建"五高五标五化"的企业级智能运维矩阵。


### 1.2 核心设计理念

| 理念 | 说明 | 实现方式 |
|------|------|----------|
| **五高架构** | 高可用、高性能、高安全、高扩展、高智能 | 模块化设计、WebSocket 实时、安全沙箱 |
| **五标体系** | 标准化、规范化、自动化、可视化、智能化 | 统一类型系统、自动化测试、可视化监控 |
| **五化转型** | 流程化、数字化、生态化、工具化、服务化 | 工作流引擎、数据持久化、插件架构 |
| **五维评估** | 时间维、空间维、属性维、事件维、关联维 | 全链路追踪、多维数据分析 |

### 3.2 路由架构

```typescript
// 路由配置 (src/app/routes.ts)
// 使用 createHashRouter 实现 SPA 路由

export const router = createHashRouter([
  {
    path: "/",
    Component: Layout,           // 根布局组件
    children: [
      // ─── 核心监控 ───
      { index: true, Component: DataMonitoring },      // 默认首页
      { path: "follow-up", Component: FollowUpPanel }, // 一键跟进
      { path: "patrol", Component: PatrolDashboard },  // 巡查仪表板
      { path: "operations", Component: OperationCenter }, // 操作中心
      
      // ─── AI Family ───
      { path: "ai-family", Component: AIFamilyPage },      // AI Family 入口
      { path: "ai-family/:subpage", Component: AIFamilyRouter }, // 子页面路由
      
      // ─── 开发工具 ───
      { path: "files", Component: LocalFileManager },   // 文件管理
      { path: "terminal", Component: CLITerminal },     // 终端
      { path: "ide", Component: IDEPanel },             // IDE
      { path: "database", Component: DatabaseManager }, // 数据库
      
      // ─── 系统设置 ───
      { path: "settings", Component: SystemSettings },  // 系统设置
      { path: "models", Component: ModelProviderPanel }, // 模型配置
      { path: "theme", Component: ThemeCustomizer },    // 主题
      
      // ─── 404 ───
      { path: "*", Component: NotFound },
    ],
  },
]);
```

### 7.2 命名双标准

```typescript
// 前端 UI 层: camelCase (高频渲染优化)
interface NodeData {
  gpu: number;      // GPU 利用率
  mem: number;      // 内存利用率
  temp: number;     // 温度
}

// DB 层: snake_case (与 PostgreSQL 列名一致)
interface NodeStatusRecord {
  gpu_util: number;
  mem_util: number;
  temp_celsius: number;
}

// 转换函数
export function toNodeData(record: NodeStatusRecord): NodeData {
  return {
    gpu: record.gpu_util,
    mem: record.mem_util,
    temp: record.temp_celsius,
  };
}
```

```typescript
// electron/main.ts 安全配置

mainWindow = new BrowserWindow({
  webPreferences: {
    nodeIntegration: false,      // 禁用 Node.js 集成

mainWindow.webContents.setWindowOpenHandler(({ url }) => {
```
