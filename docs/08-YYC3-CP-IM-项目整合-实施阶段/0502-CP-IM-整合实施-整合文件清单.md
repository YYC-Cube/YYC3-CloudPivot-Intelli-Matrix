## 🔧 需要的代码改造

### text-to-sql.ts 改造要点

```typescript
// 1. 移除 Supabase 依赖
- import { supabase } from "./supabaseClient";
+ import { storage } from "./storage/abstract";

// 2. 改造数据库执行逻辑
- const result = await supabase.rpc('execute_sql', { sql });
+ const result = await storage.execute(sql);

// 3. 调整错误处理
- if (error) { console.error('[Supabase]', error); }
+ if (error) { console.error('[Storage]', error); }
```

### 导入路径调整
