# 01-YYC³-CP-IM-启动规划阶段 - 0101-CP-IM-项目规划 - 项目章程与愿景

## 1. 项目概述

### 1.1 项目背景

YYC³ CloudPivot Intelli-Matrix (CP-IM) 是一个现代化的智能监控与运维平台，专为 YYC³ Family 内部 AI 研发与运维团队设计。项目基于 React 19.2.4 + TypeScript 5.9.3 技术栈，采用 Electron 28 实现跨平台桌面应用支持，并集成 PWA 技术提供离线能力。

### 1.2 项目定位

- **核心定位**：AI 驱动的智能监控与运维平台
- **目标用户**：AI 研发团队、运维工程师、技术管理者
- **应用场景**：实时监控、智能运维、数据可视化、跨平台管理
- **技术特色**：赛博朋克设计、实时 WebSocket 推送、AI 智能辅助、PWA 离线支持

### 1.3 项目愿景

打造一个具备自我感知、自我决策、自我执行能力的智能运维平台，实现人机共生的协作模式，通过 AI 技术提升运维效率，降低运维成本，保障系统稳定性。

#### 2.2.2 组件架构

- **业务组件**：60+ 功能组件（Dashboard, OperationCenter, PatrolDashboard 等）
- **UI 组件库**：40+ Radix UI 原语组件
- **自定义 Hooks**：20+ 复用逻辑（useWebSocketData, useI18n, usePatrol 等）
- **Context Providers**：全局状态管理（AuthContext, I18nContext, WebSocketContext, ViewContext）

### 2.3 核心技术特性

#### 2.3.1 类型安全
- **集中式类型定义**：`src/app/types/index.ts`（1000+ 行）
- **严格 TypeScript 模式**：零 `any` 类型
- **类型转换函数**：DB Schema → UI 类型转换

#### 2.3.2 实时通信
- **WebSocket 连接**：ws://localhost:3113/ws
- **消息类型**：qps_update, latency_update, node_status, alert
- **降级机制**：连接失败时切换到本地模拟数据
- **节流控制**：100ms UI 更新节流

#### 2.3.3 PWA 离线支持
- **Service Worker**：离线缓存
- **Background Sync**：后台同步队列
- **Install Prompt**：PWA 安装引导
- **离线模式**：无网络也能使用

#### 2.3.4 跨平台支持
- **Web 应用**：支持所有现代浏览器
- **Electron 桌面**：macOS, Windows, Linux
- **响应式设计**：完美适配桌面、平板、移动端
