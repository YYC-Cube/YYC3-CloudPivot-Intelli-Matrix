## 📝 脚本系统说明

### 7.1 智能检测系统

#### 核心功能

智能检测系统是一个全面的生产部署前检测工具集，旨在确保 YYC³ CloudPivot Intelli-Matrix 项目的多维度质量保证。

**特性**:
- 多维度检测（环境、功能、存储、API、Ollama）
- 智能协同（自动化检测流程）
- 实用高效（轻量级设计）
- 灵活配置（自定义检测选项）
- 详细报告（多格式报告生成）

#### 检测模块

1. **环境可用性检测** (`detect-environment.ts`)
   - Node.js 版本检查
   - pnpm 包管理器可用性
   - 操作系统平台
   - 本地存储可用性
   - 网络连接状态

2. **功能完整性检测** (`detect-functionality.ts`)
   - 核心功能模块
   - 用户交互功能
   - 数据处理功能
   - API 集成功能

3. **本地存储检测** (`detect-storage.ts`)
   - LocalStorage 可用性
   - IndexedDB 可用性
   - SessionStorage 可用性
   - Cookie 可用性

4. **大模型 API 检测** (`detect-llm-api.ts`)
   - OpenAI API Key 配置
   - 自定义 Base URL 配置
   - 网络连接测试
   - 模型列表获取

5. **Ollama 本地检测** (`detect-ollama.ts`)
   - Ollama URL 配置
   - Ollama 服务状态
   - 模型列表获取

#### 使用方式

**浏览器控制台**:
```javascript
// 快速检测
await SmartDetect.run();

// 完整检测
await SmartDetect.fullCheck({
  testLLM: true,
  testOllama: true,
  saveReport: true,
  reportFormat: "md"
});
```

**命令行**:
```bash
# 快速检测
pnpm tsx scripts/smart-detect.ts quick

# 完整检测
pnpm tsx scripts/smart-detect.ts full --test-llm --test-ollama
```
