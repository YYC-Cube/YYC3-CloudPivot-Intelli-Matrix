# YYC³-CP-IM-用户指南-系统管理-系统诊断

## 功能概述

系统诊断提供全面的系统健康检查，自动发现潜在问题并提供修复建议。

---

## 诊断项目

### 1. 硬件诊断
- CPU 健康检查
- 内存健康检查
- 磁盘健康检查
- 网卡健康检查

### 2. GPU 诊断
- GPU 温度检测
- GPU 使用率检测
- 显存使用检测
- GPU 错误检测

### 3. 系统诊断
- 操作系统健康
- 内核版本检查
- 驱动版本检查
- 系统资源检查

### 4. 服务诊断
- 服务运行状态
- 服务依赖检查
- 服务配置检查
- 服务日志检查

---

## 操作步骤

### 执行全面诊断

1. 进入"系统诊断"页面
2. 点击"开始诊断"按钮
3. 等待诊断完成（通常 3-5 分钟）
4. 查看诊断报告

### 查看诊断结果

1. 诊断完成后，显示诊断报告
2. 报告包含：
   - 系统健康评分
   - 发现的问题
   - 修复建议
   - 优先级排序

### 执行修复

1. 在诊断报告中，找到可自动修复的问题
2. 点击"一键修复"按钮
3. 确认修复操作
4. 等待修复完成

---

## 常见问题

**Q: 诊断耗时太长？**
A: 可以选择诊断范围，只诊断问题模块

**Q: 修复失败怎么办？**
A: 查看修复日志，了解失败原因，然后手动修复
