# YYC³-CP-IM-用户指南-系统设置-网络设置

## 功能概述

网络设置提供网络连接配置、节点拓扑管理、网络监控等功能。

---

## 配置项

### 1. 网络连接配置
- WebSocket 端点
- 数据库地址
- 网络连接状态

### 2. 节点拓扑
- 主节点（M4 Max）
- 辅助节点（iMac）
- 存储节点（NAS）

### 3. 网络监控
- 在线/离线状态
- 网络延迟
- 网络带宽
- 丢包率

---

## 操作步骤

### 配置网络

1. 进入"系统管理" → "系统设置" → "网络设置"
2. 点击"打开网络配置面板"按钮
3. 选择配置方式：
   - 自动检测
   - WiFi 配置
   - 手动配置
4. 填写网络参数
5. 点击"应用"按钮

### 查看节点拓扑

1. 在网络设置页面，查看节点拓扑图
2. 显示所有节点及其连接关系
3. 节点状态用颜色标识：
   - 绿色：在线
   - 红色：离线

### 测试网络连接

1. 点击"测试连接"按钮
2. 系统测试所有节点的连通性
3. 显示测试结果

## 常见问题

**Q: 节点显示离线怎么办？**
A: 检查节点网络连接和防火墙设置

**Q: 如何修改 WebSocket 端点？**
A: 在网络连接配置中，点击编辑图标，输入新的端点地址
