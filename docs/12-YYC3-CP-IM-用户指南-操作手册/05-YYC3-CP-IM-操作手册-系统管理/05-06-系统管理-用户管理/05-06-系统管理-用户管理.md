# YYC³-CP-IM-用户指南-系统管理-用户管理

## 功能概述

用户管理提供用户账号、角色、权限的统一管理，包括用户创建、角色分配、权限控制等功能。

---

## 功能特性

### 1. 用户管理
- 创建用户
- 编辑用户
- 删除用户
- 启用/禁用用户

### 2. 角色管理
- 创建角色
- 编辑角色
- 删除角色
- 角色权限分配

### 3. 权限管理
- 权限列表
- 权限组
- 权限继承

---

## 操作步骤

### 创建用户

1. 进入"用户管理"页面
2. 点击"新建用户"按钮
3. 填写用户信息：
   - 用户名
   - 邮箱
   - 密码
   - 角色
4. 点击"创建"按钮

### 编辑用户

1. 找到要编辑的用户
2. 点击"编辑"按钮
3. 修改用户信息
4. 点击"保存"按钮

### 分配角色

1. 进入用户详情页面
2. 点击"角色"标签
3. 选择要分配的角色
4. 点击"保存"按钮

### 删除用户

1. 找到要删除的用户
2. 点击"删除"按钮
3. 确认删除

## 预设角色

| 角色 | 权限说明 |
|------|---------|
| 超级管理员 | 拥有所有权限 |
| 管理员 | 拥有大部分管理权限 |
| 运维人员 | 拥有运维相关权限 |
| 开发人员 | 拥有开发相关权限 |
| 只读用户 | 只有查看权限 |

## 常见问题

**Q: 用户密码忘记了怎么办？**
A: 管理员可以重置用户密码

**Q: 如何批量导入用户？**
A: 点击"导入用户"按钮，上传 CSV 文件
