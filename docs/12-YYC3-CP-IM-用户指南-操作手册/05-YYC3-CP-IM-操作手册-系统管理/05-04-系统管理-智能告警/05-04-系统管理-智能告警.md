# YYC³-CP-IM-用户指南-系统管理-智能告警

## 功能概述

智能告警提供基于 AI 的异常检测和告警推送，及时发现系统异常并通知相关人员。

---

## 告警类型

### 1. 硬件告警
- CPU 使用率过高
- 内存使用率过高
- 磁盘空间不足
- 网络连接异常

### 2. GPU 告警
- GPU 温度过高
- GPU 使用率过高
- 显存不足
- GPU 离线

### 3. 应用告警
- 服务不可用
- 响应时间过长
- 错误率过高
- QPS 下降

### 4. 安全告警
- 异常登录
- 权限提升
- 数据泄露
- 攻击检测

---

## 操作步骤

### 配置告警规则

1. 进入"智能告警"页面
2. 点击"新建规则"按钮
3. 配置规则：
   - 规则名称
   - 告警条件
   - 告警级别
   - 通知方式
4. 保存规则

### 查看告警历史

1. 点击"告警历史"标签
2. 查看所有历史告警
3. 支持筛选和搜索

### 配置通知

1. 进入"系统管理" → "系统设置" → "通知配置"
2. 配置通知渠道：
   - 邮件通知
   - Slack 通知
   - Webhook 通知
3. 设置告警阈值

|------|------|------|
| 严重 | 红色 | 立即处理 |

## 常见问题

**Q: 如何避免告警疲劳？**
A: 设置合理的阈值，启用告警合并功能

**Q: 告警通知延迟怎么办？**
A: 检查通知配置和网络连接，或切换通知渠道
