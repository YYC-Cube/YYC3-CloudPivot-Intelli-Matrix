# YYC³-CP-IM-用户指南-系统管理-安全监控

## 功能概述

安全监控提供全方位的安全防护，包括入侵检测、漏洞扫描、合规检查等功能。

---

## 功能特性

### 1. 入侵检测
- 异常登录检测
- 暴力破解检测
- 权限提升检测
- 数据泄露检测

### 2. 漏洞扫描
- 系统漏洞扫描
- 应用漏洞扫描
- 依赖漏洞扫描
- 配置漏洞扫描

### 3. 合规检查
- 等保合规检查
- 行业合规检查
- 内部合规检查
- 合规报告生成

---

## 操作步骤

### 查看安全事件

1. 进入"安全监控"页面
2. 显示所有安全事件
3. 事件包含：
   - 事件时间
   - 事件类型
   - 事件级别
   - 事件详情

### 执行漏洞扫描

1. 点击"开始扫描"按钮
2. 选择扫描类型：
   - 快速扫描
   - 完整扫描
   - 自定义扫描
3. 点击"确认"开始扫描
4. 等待扫描完成

### 查看合规报告

1. 点击"合规报告"标签
2. 查看合规评分
3. 查看不合规项
4. 查看修复建议

## 常见问题

**Q: 如何处理安全事件？**
A: 查看事件详情，根据建议执行修复操作

**Q: 漏洞扫描会影响性能吗？**
A: 快速扫描影响较小，完整扫描建议在低峰期执行
