# YYC³-CP-IM-用户指南-系统管理-性能报表

## 功能概述

性能报表提供全面的性能数据分析，包括趋势分析、对比分析、资源预测等功能。

---

## 报表类型

### 1. 实时报表
- 实时资源使用
- 实时性能指标
- 实时服务状态

### 2. 历史报表
- 历史趋势分析
- 历史对比分析
- 历史峰值统计

### 3. 预测报表
- 资源预测
- 性能预测
- 容量规划

---

## 操作步骤

### 查看实时报表

1. 进入"性能报表"页面
2. 点击"实时报表"标签
3. 查看实时数据：
   - CPU 使用率
   - 内存使用率
   - GPU 使用率
   - 网络流量

### 查看历史报表

1. 点击"历史报表"标签
2. 选择时间范围：
   - 最近 1 小时
   - 最近 24 小时
   - 最近 7 天
   - 自定义范围
3. 查看趋势图表
4. 支持缩放和平移

### 导出报表

1. 点击"导出报表"按钮
2. 选择报表类型
3. 选择时间范围
4. 选择导出格式（PDF/Excel/CSV）
5. 点击"确认导出"

---

## 报表指标

| 指标 | 说明 | 单位 |
|------|------|------|
| CPU 使用率 | CPU 占用比例 | % |
| 内存使用率 | 内存占用比例 | % |
| GPU 使用率 | GPU 占用比例 | % |
| QPS | 每秒查询数 | req/s |
| 响应时间 | 请求响应时间 | ms |
| 错误率 | 错误请求比例 | % |

---

## 常见问题

**Q: 报表数据延迟？**
A: 检查数据采集服务的运行状态

**Q: 如何自定义报表？**
A: 点击"新建报表"按钮，配置报表参数
