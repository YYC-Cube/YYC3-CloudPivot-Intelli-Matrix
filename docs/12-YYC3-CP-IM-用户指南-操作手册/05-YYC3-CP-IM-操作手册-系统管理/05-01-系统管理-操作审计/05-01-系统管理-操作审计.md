# YYC³-CP-IM-用户指南-系统管理-操作审计

## 功能概述

操作审计记录所有用户操作行为，提供完整的审计日志、操作追踪、合规报告等功能。

---

## 功能特性

### 1. 操作日志
- 用户登录/登出
- 配置修改
- 操作执行
- 权限变更

### 2. 操作追踪
- 按用户筛选
- 按时间筛选
- 按操作类型筛选
- 按结果筛选

### 3. 合规报告
- 操作统计
- 异常操作
- 合规分析
- 报告导出

---

## 操作步骤

### 查看审计日志

1. 进入"操作审计"页面
2. 显示所有操作日志
3. 日志包含：
   - 操作时间
   - 操作用户
   - 操作类型
   - 操作详情
   - 操作结果

### 筛选日志

1. 点击"筛选"按钮
2. 选择筛选条件：
   - 用户
   - 时间范围
   - 操作类型
   - 操作结果
3. 点击"应用筛选"

### 导出审计报告

1. 点击"导出报告"按钮
2. 选择时间范围
3. 选择报告格式（PDF/Excel/CSV）
4. 点击"确认导出"

---

## 操作类型

| 类型 | 说明 |
|------|------|
| LOGIN | 用户登录 |
| LOGOUT | 用户登出 |
| CONFIG | 配置修改 |
| OPERATION | 操作执行 |
| PERMISSION | 权限变更 |
| FILE | 文件操作 |

---

## 常见问题

**Q: 审计日志保留多久？**
A: 默认保留 90 天，可在系统设置中调整

**Q: 如何追踪特定用户？**
A: 使用筛选功能，选择用户名进行筛选
