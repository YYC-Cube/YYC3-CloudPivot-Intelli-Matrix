## 第一章：功能概述 —— 主题定制的核心能力

### 1.1 主题定制核心定位

YYC³ CP-IM 的主题定制是一个**完整的主题管理系统**，支持 OKLch + HEX 双模式颜色管理、品牌定制、字体排版、阴影效果等全方位的主题定制能力。

### 1.2 核心特性矩阵

| 特性类别 | 功能描述 | 技术实现 |
|---------|---------|---------|
| **颜色系统** | OKLch + HEX 双模式颜色管理 | ThemeCustomizer.tsx + color-utils |
| **品牌定制** | Logo、系统名称、标语、背景图片 | BrandingConfig |
| **颜色定制** | 主色、次色、强调色、背景色等 | ThemeColors |
| **环形区域** | 图表颜色、侧边栏颜色 | Chart Colors + Sidebar Colors |
| **字体排版** | 无衬线字体、衬线字体、等宽字体 | ThemeTypography |
| **阴影圆角** | 阴影效果、圆角半径、透明度 | ThemeShadow |
| **预设系统** | 多套预设主题，支持搜索 | THEME_PRESETS |
| **导入导出** | 主题配置文件导入导出 | JSON 格式 |
| **实时预览** | 实时预览主题效果 | React State |

### 9.2 高级功能

#### 9.2.1 OKLch 颜色模式

**说明**：
- OKLch 是一种感知均匀的颜色空间
- 支持亮度（L）、彩度（C）、色相（h）三个维度
- 更适合人眼感知的颜色调整

**技术实现**：
```typescript
import { hexToOklch, oklchToHex } from "./theme/color-utils";

// HEX 转 OKLch
const oklch = hexToOklch("#00d4ff");
// { L: 0.5, C: 0.2, h: 200 }

// OKLch 转 HEX
const hex = oklchToHex(0.5, 0.2, 200);
// "#00d4ff"
```

#### 9.2.2 亮度调整

**说明**：
- 支持基于亮度的主题调整
- 亮度范围：0-100
- 实时调整主题亮度

**技术实现**：
```typescript
const adjustedPrimary = (() => {
  const oklch = hexToOklch(colors.primary);
  const newL = Math.max(0.05, Math.min(0.95, lightness / 100));
  return oklchToHex(newL, oklch.C, oklch.h);
})();
```
