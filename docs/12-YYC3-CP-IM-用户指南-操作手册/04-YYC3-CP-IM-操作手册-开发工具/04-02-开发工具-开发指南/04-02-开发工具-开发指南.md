### 2.2 项目安装

#### 2.2.1 克隆项目

```bash
git clone https://github.com/YYC-Cube/YYC3-CloudPivot-Intelli-Matrix.git
cd YYC3-CloudPivot-Intelli-Matrix
```

#### 2.2.2 安装依赖

```bash
pnpm install
```

#### 2.2.3 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 文件，填入必要的配置
```

```bash
# Supabase 认证
VITE_SUPABASE_URL=your_supabase_url

VITE_WS_URL=ws://localhost:3113/ws
```

### 2.4 常见问题

#### 2.4.1 依赖安装失败

**问题**：`pnpm install` 失败

**解决方案**：
```bash
# 清理缓存
pnpm store prune

# 重新安装
rm -rf node_modules pnpm-lock.yaml
pnpm install
```

#### 2.4.2 端口被占用

**问题**：端口 3218 被占用

**解决方案**：
```bash
# 查找占用端口的进程
lsof -i :3218

# 杀死进程
kill -9 <PID>

# 或者修改 Vite 配置中的端口
```

#### 2.4.3 TypeScript 类型错误

**问题**：TypeScript 类型检查失败

**解决方案**：
```bash
# 重新生成类型
pnpm type-check

# 清理缓存
pnpm clean

# 重新安装依赖
pnpm install
```

## 第四章：开发规范 —— 代码质量保障

### 4.1 代码风格

#### 4.1.1 ESLint 配置

**配置文件**：`eslint.config.js`

**核心规则**：
- TypeScript 严格模式
- React Hooks 规则
- 无 `any` 类型
- 无未使用的变量（前缀 `_`）

**运行检查**：
```bash
# 检查代码
pnpm lint

# 自动修复
pnpm lint:fix
```

#### 4.1.2 Prettier 配置

**配置文件**：`.prettierrc.json`

**核心规则**：
- 分号：必需
- 引号：双引号
- 缩进：2 空格
- 行宽：100
- 换行：LF

**格式化代码**：
```bash
# 格式化所有文件
pnpm prettier --write .
```

---

### 4.2 提交规范

#### 4.2.1 Commit Message 规范

**格式**：
```
<type>(<scope>): <subject>

<body>

<footer>
```

**类型（type）**：
- `feat`：新功能
- `fix`：修复 bug
- `docs`：文档更新
- `style`：代码格式（不影响功能）
- `refactor`：重构
- `test`：测试相关
- `chore`：构建/工具相关

**示例**：
```
feat(dashboard): add real-time data monitoring

- Add WebSocket connection
- Implement data streaming
- Add chart visualization

Closes #123
```

---

### 4.3 命名规范

#### 4.3.1 文件命名

| 类型 | 命名规范 | 示例 |
|------|---------|------|
| 组件文件 | PascalCase | `Dashboard.tsx` |
| 工具文件 | camelCase | `utils.ts` |
| 类型文件 | camelCase | `types.ts` |
| 样式文件 | kebab-case | `styles.css` |
| 测试文件 | PascalCase.test.tsx | `Dashboard.test.tsx` |

#### 4.3.2 变量命名

| 类型 | 命名规范 | 示例 |
|------|---------|------|
| 变量 | camelCase | `const userName = 'John'` |
| 常量 | UPPER_SNAKE_CASE | `const MAX_RETRY_COUNT = 3` |
| 函数 | camelCase | `function getUserData()` |
| 类 | PascalCase | `class UserManager {}` |
| 接口 | PascalCase | `interface UserData {}` |
| 类型 | PascalCase | `type UserRole = 'admin' | 'user'` |

**响应**：
```json
{
  "nodes": [
    {
      "id": "GPU-A100-01",
      "name": "GPU-A100-01",
      "status": "normal",
      "metrics": {
        "gpuUsage": 85.5,
        "memoryUsage": 78.2,
        "temperature": 65.3
      }
    }
  ],
  "timestamp": 1640995200000
}
```

#### 7.1.2 获取历史数据

**端点**：`GET /api/monitoring/history`

**参数**：
- `startTime`：开始时间戳
- `endTime`：结束时间戳
- `interval`：时间间隔

**响应**：
```json
{
  "data": [
    {
      "timestamp": 1640995200000,
      "gpuUsage": 85.5,
      "memoryUsage": 78.2
    }
  ]
}
```

```json
  "result": {
    "success": true,
    "message": "Node restarted successfully"
  }
}
```
