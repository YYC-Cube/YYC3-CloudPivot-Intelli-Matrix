## 第一章：功能概述 —— Web 端 CLI 终端模拟器

### 1.1 终端核心定位

YYC³ CP-IM 的终端集成是一个**Web 端 CLI 终端模拟器**，提供完整的命令行交互体验，支持 yyc3 命令、自动补全、历史导航等功能。

| **自动补全** | Tab 补全、命令提示、参数补全 | 自动补全算法 |

export function useTerminal({ onNavigate }: { onNavigate: (path: string) => void }) {
  const [history, setHistory] = useState<TerminalEntry[]>([]);
  const [inputValue, setInputValue] = useState("");
  const [completions, setCompletions] = useState<string[]>([]);

  const execute = (command: string) => {
    // 执行命令逻辑
    const output = processCommand(command);

}

### 3.3 历史导航功能

#### 3.3.1 命令历史记录

**功能特性**：
- 📚 **历史记录**：记录所有执行的命令
- ⬆️ **向上导航**：按 `↑` 键浏览历史命令
- ⬇️ **向下导航**：按 `↓` 键浏览历史命令

**使用方式**：
1. 按 `↑` 键查看上一个命令
2. 按 `↓` 键查看下一个命令
3. 修改命令后按 `Enter` 执行

**技术实现**：
```typescript
const handleHistoryNav = (direction: "up" | "down") => {
  const historyCommands = history.filter((h) => h.input).map((h) => h.input!);
  if (direction === "up" && historyIndex < historyCommands.length - 1) {
    setHistoryIndex(historyIndex + 1);
    setInputValue(historyCommands[historyCommands.length - 1 - historyIndex - 1]);
  } else if (direction === "down" && historyIndex > 0) {
    setHistoryIndex(historyIndex - 1);
    setInputValue(historyCommands[historyCommands.length - 1 - historyIndex + 1]);
  }
};
```

A:
- 检查命令是否正确
